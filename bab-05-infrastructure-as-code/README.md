# Bab 5 — Infrastructure as Code

## Isi Folder
- `README.md`
- `main.tf`
- `variables.tf`
- `outputs.tf`
- `terraform.tfvars.example`

## Objek Kajian
Objek kajian pada bab ini adalah **konfigurasi Terraform minimal** untuk provisioning resource AWS.

## Resource yang dicontohkan
- 1 Security Group
- 1 EC2 Instance

## Catatan
Contoh ini bersifat edukatif. Sebelum dijalankan, pengguna perlu:
- mengonfigurasi AWS CLI / credentials,
- menyesuaikan AMI,
- menyesuaikan region,
- memahami potensi biaya resource cloud.
