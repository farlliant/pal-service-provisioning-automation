# Bab 6 — CI/CD Pipeline Management

## Isi Folder
- `README.md`
- `Jenkinsfile`
- `deployment_notes.md`

## Objek Kajian
Objek kajian pada bab ini adalah **pipeline Jenkins sederhana** untuk alur checkout, build, test, package, dan deploy.

## Hal yang bisa diamati
- pipeline ditulis sebagai kode,
- stage dipisahkan dengan jelas,
- ada post action untuk notifikasi hasil,
- alur kerja lebih mudah ditinjau di repository.

## Catatan
Perintah di Jenkinsfile dibuat generik agar mudah disesuaikan dengan proyek lain.
