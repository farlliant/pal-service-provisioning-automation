# Bab 4 — Manajemen Konfigurasi

## Isi Folder
- `README.md`
- `inventory.ini`
- `deploy_webserver.yml`

## Objek Kajian
Objek kajian pada bab ini adalah **contoh inventory dan playbook Ansible** untuk menstandarkan konfigurasi web server.

## Hal yang bisa diamati
- daftar host disimpan terpisah di inventory,
- task ditulis dalam bentuk playbook,
- package, file, dan service bisa dikelola dari satu tempat.

## Tujuan
Menunjukkan bahwa konfigurasi banyak host dapat dilakukan lebih konsisten dibandingkan login dan konfigurasi manual satu per satu.
