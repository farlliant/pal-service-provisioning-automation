# Bab 1 — Pengantar Penyediaan dan Automasi Layanan

## Isi Folder
- `README.md` → penjelasan singkat objek kajian
- `provision_nginx_example.sh` → contoh skrip provisioning sederhana

## Objek Kajian
Objek kajian pada bab ini adalah **skrip provisioning sederhana** yang menyiapkan web server Nginx pada Ubuntu/Debian.

## Mengapa dipilih
Karena provisioning adalah tahap awal sebelum aplikasi dijalankan. Contoh ini menunjukkan bahwa server bisa disiapkan dengan langkah berulang yang ditulis sebagai skrip, bukan dikerjakan manual satu per satu.

## Hal yang bisa diamati
- update package index,
- instalasi Nginx,
- aktivasi service,
- pengecekan status service.

## Catatan
Skrip ini cocok dijadikan contoh konsep, bukan konfigurasi produksi penuh.
